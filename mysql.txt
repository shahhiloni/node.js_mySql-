## ----------------- how SQL works 
1. used for write SQL Query 

-- EX : SELECT * FROM students;

2. SQL engine process the query 

- check the syntax, validation, permission, security, controls, access,....

3. if everything going floe than database execute query 


Basic Commands of SQL 
1. Create table 
- Data definition language 

CREATE TABLE students (
    id INT, 
    name  VARCHAR(200), 
    course VARCHAR(50), 
    marks INT, 
    Age INT
)

2. INSERT DATA 
-  Data manipulation language
INSERT INTO students VALUES (1, 'xxx', 'MERN STACK', 85, 22);

3. FETCH DATA

SELECT name, marks FROM students

4. UPDATE DATA 
-  Data manipulation language
UPDATE students SET marks = 90 WHERE id = 1;

5. DELETE DATA 

DELETE FROM students WHERE id = 1;

// Foreign key : building relation between two table or more than two tables 
- Note: only For relational database

- Foreign_key : valid data or actual data and never null 
Note : it can be null when data is not exist

child table : students 

relation : register.id -> students.user_id
